**Stw1ter** — это современный мессенджер, разрабатываемый как открытое программное обеспечение. На данный момент реализован общий чат для всех пользователей. В планах — расширенная функциональность: NFT-аутентификация, уникальные аватары, токенизированные сообщения, личные чаты и приватные комнаты.

Мессенджер создан на: HTML, CSS, JS, Python(flask), пока что только на этом.

Реализовано:
Общий чат (глобальная комната)
Система регистрации и аутентификации
Отправка и получение сообщений в реальном времени

В планах:

Интеграция NFT (вход по токену, NFT-аватары)

Личные сообщения и приватные комнаты
Сквозное шифрование сообщений
Адаптивный веб-интерфейс

Технологии:
Backend: Go (Fiber, WebSockets)
База данных: PostgreSQL / SQLite
NFT: EVM-совместимые блокчейны
